# 📝 toDoList

Aplicação web de lista de tarefas desenvolvida com **React** e **Firebase**, permitindo autenticação de usuários e gerenciamento completo de tarefas.

## 🚀 Funcionalidades

* 🔐 Cadastro de usuário
* 🔑 Login com autenticação via Firebase
* 👤 Página administrativa (Admin)
* ➕ Criar novas tarefas
* ✏️ Editar tarefas existentes
* ✅ Concluir tarefa (remove da lista)
* 🔄 Atualização em tempo real com Firebase

---

## 🛠️ Tecnologias Utilizadas

* **React**
* **Firebase Authentication**
* **Firestore Database**
* **CSS**
